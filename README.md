AllJew is a modern Jewish daily life utility app built with **React Native and Expo**. The app is designed to bring together essential tools used in everyday Jewish life into a single clean and intuitive mobile experience.

Rather than relying on multiple apps or websites, AllJew provides a centralized interface for viewing **zmanim (halachic times)**, tracking **sunrise and sunset**, accessing **daily davening tools**, finding **kosher locations nearby**, and using a **Jewish directional compass**.

The goal of the project is to combine thoughtful UX design with practical religious functionality, creating a reliable tool that can be used daily by members of the Jewish community.


### Core Features

• **Zmanim Calculations** – Accurate halachic times based on the user’s location
• **Sunrise & Sunset Tracking** – Automatically calculated using geolocation
• **Daily Davening Tools** – Quick access tools designed to support daily prayer routines
• **Kosher Location Finder** – Uses the Google Places API to help users find nearby kosher food and Jewish resources
• **Jewish Compass** – Directional compass designed to help orient users for prayer
• **Location-Based Services** – Dynamic time calculations based on where the user is located
• **Premium Subscription System** – Built with RevenueCat to support future premium features
• **Minimalist UI Design** – Clean and modern interface optimized for daily usability

## 🧠 Architecture

The app is built using a modern React Native architecture with a focus on modular components and scalable data handling.

Key architectural decisions include:

- **TypeScript-first development** to improve reliability and maintainability
- **Location-based calculations** using geolocation and timezone lookup
- **API-driven features** such as the kosher location finder using Google Places
- **Subscription management** handled through RevenueCat
- **Component-based UI structure** to allow rapid feature iteration

This structure allows the app to remain lightweight while still supporting complex location-based calculations and future feature expansion.

---

## 💡 Purpose

I built AllJew to combine clean technology design with meaningful daily Jewish functionality. Many existing tools for zmanim, kosher discovery, and prayer direction are spread across multiple platforms or older interfaces.

This project aims to create a modern, reliable, and production-ready mobile application that simplifies those tools into one intuitive interface. By leveraging modern mobile technologies like React Native and Expo, the app can deliver a fast and responsive experience while remaining easy to maintain and expand.
